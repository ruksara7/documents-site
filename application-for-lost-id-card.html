Application for Lost School ID Card

If a student loses their school ID card, they must write an application to the principal requesting a new ID card. Below are simple copy-ready application formats in English and Hindi.

English Application

To The Principal [School Name] Date: ___ / ___ / 20__ Subject: Application for Lost School ID Card Respected Sir/Madam, I am a student of class ______ in your school. Unfortunately, I have lost my school ID card. Despite searching for it carefully, I have not been able to find it. Therefore, I kindly request you to issue me a new school ID card. Thank you. Yours sincerely Name: Class: Roll Number:

Hindi Application

सेवा में, प्रधानाचार्य महोदय / महोदया [विद्यालय का नाम] दिनांक: ___ / ___ / 20__ विषय: खोए हुए स्कूल पहचान पत्र के लिए आवेदन महोदय / महोदया, मैं आपके विद्यालय की कक्षा ______ का छात्र / छात्रा हूँ। मेरा स्कूल पहचान पत्र (ID कार्ड) कहीं खो गया है। मैंने उसे बहुत खोजने का प्रयास किया लेकिन वह नहीं मिला। अतः आपसे विनम्र निवेदन है कि कृपया मुझे नया स्कूल पहचान पत्र जारी करने की कृपा करें। धन्यवाद। भवदीय नाम: कक्षा: रोल नंबर:

Quick Tips

  • Mention your class and roll number clearly.
  • Write the correct school name.
  • Keep the application polite and simple.

Documents Required

  • Application letter
  • School record verification
  • Replacement fee if required
